Description: Public instance for Desktop, Files/Folders and Empty Area.
	PROCEED AT YOUR OWN RISK! The developer is not responsible for any damage to your computer.

Cast_To_Device 				- Disable/Enable "Cast to Device" option.
Command_Promt_ADMIN 		- Disable/Enable "Open terminal (Administrator)" option. Command prompt with administrative privileges.
Command_Promt_here 			- Disable/Enable "Open terminal here" option. Open this folder in Command Promt.
Give_Access_To 				- Disable/Enable "Give access to" option.
New_option 					- Disable/Enable "New" option. Important: To create folders use Shift+Ctrl+N hotkey.
Restore_Previous_Versions	- Disable/Enable "Restore previous versions" option.
Shop_For_Music_Online		- Disable/Enable "Shop For Music Online" option.
Send_To						- Disable/Enable "Send to" option. Use EDIT_THIS_MENU.cmd file, to edit this menu.
Share						- Disable/Enable "Share" option. NOT WORKING!

Cast_To_Device 				- Отключить/Включить опцию "Отправить на устройство".
Command_Promt_ADMIN 		- Отключить/Включить опцию "Открыть терминал (Администратор)". Командная строка с повышенными привилегиями.
Command_Promt_here 			- Отключить/Включить опцию "Открыть терминал здесь". Открыть данную папку в Командной строке.
Give_Access_To 				- Отключить/Включить опцию "Открыть доступ для".
New_option 					- Отключить/Включить опцию "Создать". Важно: Для создания папок используйте сочетание клавиш Shift+Ctrl+N.
Restore_Previous_Versions	- Отключить/Включить опцию "Восстановить предыдущие версии".
Shop_For_Music_Online		- Отключить/Включить опцию "Искать музыку в интернете".
Send_To						- Отключить/Включить опцию "Отправить". Используйте EDIT_THIS_MENU.cmd, чтобы редакторовать меню.
Share						- Отключить/Включить опцию "Поделиться". НЕ РАБОТАЕТ!
